# @o-w-o/ui-ext--formily

## Form 基础能力和组件


## 功能

1. 提供 @material-ui 与 @formily/react 和 @formily/react-schema-render 的整合；
1. 提供 基于 Form 的，基础的 Form/FormItem(s) 组件和扩展接口和方法；
1. 提供 基于 SchemaForm 的，基础的 SchemaForm/Field(s) 组件和扩展接口和方法；
1. 提供 布局功能。
